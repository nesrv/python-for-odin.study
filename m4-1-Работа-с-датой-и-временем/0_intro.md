## **2. –ú–æ–¥—É–ª—å `time`**

### **–û—Å–Ω–æ–≤–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏**

- **`time.time()`** ‚Üí –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Ç–µ–∫—É—â–µ–µ –≤—Ä–µ–º—è –≤ —Å–µ–∫—É–Ω–¥–∞—Ö —Å –Ω–∞—á–∞–ª–∞ —ç–ø–æ—Ö–∏ (Unix-–≤—Ä–µ–º—è).
- **`time.sleep(secs)`** ‚Üí –ü—Ä–∏–æ—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ—Ç –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ –ø—Ä–æ–≥—Ä–∞–º–º—ã –Ω–∞ `secs` —Å–µ–∫—É–Ω–¥.
- **`time.ctime([secs])`** ‚Üí –ü—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç –≤—Ä–µ–º—è –≤ —Å–µ–∫—É–Ω–¥–∞—Ö –≤ —Å—Ç—Ä–æ–∫—É —Ñ–æ—Ä–º–∞—Ç–∞ `"Day Mon DD HH:MM:SS YYYY"`. –ï—Å–ª–∏ `secs` –Ω–µ —É–∫–∞–∑–∞–Ω–æ, –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —Ç–µ–∫—É—â–µ–µ –≤—Ä–µ–º—è.
- **`time.localtime([secs])`** ‚Üí –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç struct_time (–∫–æ—Ä—Ç–µ–∂ —Å –∞—Ç—Ä–∏–±—É—Ç–∞–º–∏) –¥–ª—è –ª–æ–∫–∞–ª—å–Ω–æ–≥–æ –≤—Ä–µ–º–µ–Ω–∏.
- **`time.mktime(struct_time)`** ‚Üí –û–±—Ä–∞—Ç–Ω–∞—è –æ–ø–µ—Ä–∞—Ü–∏—è –¥–ª—è `localtime()`: –ø—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç struct_time –≤ —Å–µ–∫—É–Ω–¥—ã —Å –Ω–∞—á–∞–ª–∞ —ç–ø–æ—Ö–∏.
- **`time.gmtime([secs])`** ‚Üí –ê–Ω–∞–ª–æ–≥ `localtime()`, –Ω–æ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç UTC (–≤—Ä–µ–º—è –ø–æ –ì—Ä–∏–Ω–≤–∏—á—É).
- **`time.strptime(string, format)`** ‚Üí –†–∞–∑–±–∏—Ä–∞–µ—Ç —Å—Ç—Ä–æ–∫—É –≤—Ä–µ–º–µ–Ω–∏ –≤ struct_time —Å–æ–≥–ª–∞—Å–Ω–æ —Ñ–æ—Ä–º–∞—Ç—É.
- **`time.strftime(format[, struct_time])`** ‚Üí –§–æ—Ä–º–∞—Ç–∏—Ä—É–µ—Ç struct_time –≤ —Å—Ç—Ä–æ–∫—É (–µ—Å–ª–∏ –Ω–µ —É–∫–∞–∑–∞–Ω–æ, –∏—Å–ø–æ–ª—å–∑—É–µ—Ç `localtime()`).
- **`time.asctime([struct_time])`** ‚Üí –ê–Ω–∞–ª–æ–≥ `ctime()`, –Ω–æ –ø—Ä–∏–Ω–∏–º–∞–µ—Ç struct_time (–ø–æ —É–º–æ–ª—á–∞–Ω–∏—é ‚Äî —Ç–µ–∫—É—â–µ–µ –≤—Ä–µ–º—è).

```python
import time

# 1. –¢–µ–∫—É—â–µ–µ –≤—Ä–µ–º—è –≤ —Å–µ–∫—É–Ω–¥–∞—Ö —Å 1.01.1970 (Unix-—ç–ø–æ—Ö–∞)

# 2. –ü—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –≤ —á–∏—Ç–∞–µ–º—ã–π —Ñ–æ—Ä–º–∞—Ç

# 4. Struct_time (—Ä–∞–∑–±–∏–≤–∫–∞ –≤—Ä–µ–º–µ–Ω–∏ –Ω–∞ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã)
```
