### Входные данные:

красный


```python
r, g, b = map(int, input('Введи цвета через пробел: ').split())

match (r, g, b):
    case (1, 0, 0):
        print("Красный")
    case (0, 1, 0):
        print("Зеленый")
    case (0, 0, 1):
        print("Синий")
    case (1, 1, 0):
        print("Желтый")
    case (1, 0, 1):
        print("Пурпурный")
    case (0, 1, 1):
        print("Бирюзовый")
    case (1, 1, 1):
        print("Белый")
    case (0, 0, 0):
        print("Черный")
    case _:
        print("Неизвестный цвет")

```
